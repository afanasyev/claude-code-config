---
name: prd
description: "Prepare a PRD structured as vertical slices — each iteration is end-to-end, testable, and deployable. No code."
---

Do NOT write or modify any code. Prepare a PRD for the task. Structure it as vertical slices — each slice is an end-to-end iteration that produces a testable, runnable result. No horizontal layers (all DB first, then all logic). Include problem, goal, and definition of done for each iteration.
