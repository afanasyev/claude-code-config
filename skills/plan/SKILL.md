name: plan
description: "No code until approved. Read context, ultrathink, ask clarifying questions, write a detailed plan listing each file and change, wait for approval."
---

Do NOT write any code until approved. Do NOT use the built-in plan mode tool. Steps:

1. Read the task and relevant files, analyze the given context.
2. Ultrathink about what changes are needed, and before executing, ask clarifying questions to identify corner cases, uncover weaknesses, and better understand the desired outcome. When providing options, include your recommendation.
3. Write a plan listing EACH file to change and WHAT to change.
4. Wait for approval. Do not start coding until approved.
